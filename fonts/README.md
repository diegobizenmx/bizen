# Feather Bold Font Files

Por favor, coloca los archivos de la fuente "Feather Bold" en esta carpeta:

- `Feather-Bold.woff2` (recomendado - formato más eficiente)
- `Feather-Bold.woff` (alternativa)
- `Feather-Bold.ttf` (fallback)

La fuente se cargará automáticamente usando Next.js font optimization desde `src/app/layout.tsx`.

## Formatos de fuente recomendados:
- **woff2**: Formato más moderno y eficiente (mejor compresión)
- **woff**: Formato ampliamente soportado
- **ttf**: Formato tradicional como fallback

## Instrucciones:
1. Descarga o copia los archivos de fuente "Feather Bold"
2. Colócalos en esta carpeta (`fonts/` en la raíz del proyecto)
3. Asegúrate de que los nombres de archivo sean exactamente:
   - `Feather-Bold.woff2`
   - `Feather-Bold.woff`
   - `Feather-Bold.ttf`

La fuente se aplicará automáticamente a toda la aplicación una vez que los archivos estén en su lugar.
